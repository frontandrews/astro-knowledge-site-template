# External Content

As colecoes editoriais nao ficam mais versionadas neste repositorio-base.

Fallback padrao para clones limpos: `examples/starter-content`

Opcao recomendada para um repositorio editorial separado: `../site-content`

Colecoes sincronizadas: articles, concepts, glossary, challenges, roadmaps

Use `pnpm content:sync` para sincronizar o conteudo configurado antes de rodar o site.
